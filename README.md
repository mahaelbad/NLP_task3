# 🧠 Toxic Content Classification + Image Captioning & Database Integration

This project is an extension of a previous NLP task focused on **toxic content classification**, enhanced with additional features including **image captioning** and **database integration**.

---

## 📌 Project Overview

The system aims to:
- Classify text into multiple toxicity categories (multi-label classification).
- Improve usability by adding a **Streamlit-based interface**.
- Extend functionality with:
  - 🖼️ Image Captioning module
  - 🗄️ Database storage for processed inputs and predictions

---

## 🚀 Features

### 🧾 1. Toxic Content Classification
- Multi-label classification model
- Handles different types of toxic comments
- Built using NLP preprocessing + transformer/ML model

### 🖼️ 2. Image Captioning Module
- Generates captions for uploaded images
- Uses deep learning-based vision-language model

### 🗄️ 3. Database Integration
- Stores:
  - User inputs
  - Model predictions
  - Processed results
- Enables future analysis and tracking

### 🌐 4. Streamlit App
- Simple interactive UI
- Supports:
  - Text input classification
  - Image upload for captioning
  - Display of results in real-time
